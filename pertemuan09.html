Apa itu Fungsi?

Fungsi adalah blok kode yang diberi nama dan dapat dipanggil berulang kali.

Fungsi membantu kita:

  • Menghemat kode (Don't Repeat Yourself / DRY)
  • Mengorganisir kode agar rapi

Analogi: Vending Machine

ü§ñ

Input: Uang + Kode Produk

PROSES (Black Box)

Output: Minuman

Mendefinisikan Fungsi


def nama_fungsi(parameter):
    # Blok kode perintah
    # ...
    return hasil  # Opsional

Contoh Sederhana


def sapa_dunia():
    print("Halo Dunia!")

# Memanggil (Call)
sapa_dunia()
# Output: Halo Dunia!

Parameter & Argumen

Kita bisa mengirim data ke dalam fungsi.


def sapa_nama(nama):
    print(f"Halo, {nama}!")

sapa_nama("Budi")   # Halo, Budi!
sapa_nama("Siti")   # Halo, Siti!

Fungsi Penjumlahan:


def tambah(a, b):
    print(f"Hasil {a} + {b} = {a + b}")

tambah(5, 3)  # Hasil 5 + 3 = 8

Return Value

print() hanya menampilkan ke layar.

return mengembalikan nilai ke pemanggil, sehingga bisa disimpan dalam variabel atau diolah lagi.


def kuadrat(x):
    return x * x

hasil = kuadrat(5)
print(hasil + 10)  
# Output: 35 (25 + 10)

Scope Variabel (Cakupan)

Global Scope

Dibuat di luar fungsi. Bisa diakses di mana saja.


x = 10  # Global

def cek():
    print(x) # Bisa

Local Scope

Dibuat di dalam fungsi. HANYA hidup di dalam fungsi.


def hitung():
    y = 5  # Local
    print(y)

hitung()
print(y) # ERROR!

Latihan: Fungsi Geometri

Buatlah fungsi-fungsi berikut:

  1. luas_persegi(sisi) → mengembalikan luas.
  2. luas_segitiga(alas, tinggi) → mengembalikan luas.

def luas_persegi(sisi):
    return sisi * sisi

print(luas_persegi(4))  # 16

Ringkasan Hari Ini

def Keyword untuk membuat fungsi.
Parameter Variabel input di definisi fungsi.
Return Mengembalikan hasil proses.
Scope Wilayah hidup variabel (Local vs Global).
