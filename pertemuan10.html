String Methods

Python memiliki banyak fungsi bawaan untuk memanipulasi teks.

upper() Kapital semua
lower() Kecil semua
title() Kapital Tiap Kata
strip() Hapus spasi di ujung
replace(a, b) Ganti a dengan b 

s = "  Hello Python  "

print(s.upper())      # "  HELLO PYTHON  "
print(s.strip())      # "Hello Python"
print(s.replace("o", "a")) 
# "  Hella Pythan  "

Mencari dalam String


text = "Belajar Python itu seru"

# Mencari posisi (index) substring
posisi = text.find("Python")
print(posisi)  # 8

# Menghitung jumlah 'a'
jumlah_a = text.count("a")
print(jumlah_a) # 1

# Cek awalan/akhiran
print(text.startswith("Belajar")) # True
print(text.endswith("susah"))     # False

String Slicing

Sama seperti List, String bisa dipotong.


kode = "PKOM-2026-A"

matkul = kode[0:4]   # "PKOM"
tahun = kode[5:9]    # "2026"
kelas = kode[-1]     # "A"

print(f"Matkul: {matkul}, Tahun: {tahun}")

String Formatting (f-string)

Cara modern menyisipkan variabel ke string.


nama = "Budi"
nilai = 95.5678

# Dasar
print(f"Halo {nama}, selamat!")

# Angka Desimal (2 digit belakang koma)
print(f"Nilai Anda: {nilai:.2f}")  # 95.57

# Pemisah Ribuan
uang = 5000000
print(f"Rp {uang:,}")  # Rp 5,000,000
