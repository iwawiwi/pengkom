Review: Struktur if-elif-else

Digunakan ketika ada lebih dari dua kemungkinan kondisi.

  • Program akan mengecek kondisi dari atas ke bawah.
  • Jika satu kondisi True, blok tersebut dijalankan dan sisanya diabaikan. 

if kondisi1:
    # Kode jika kondisi1 TRUE
elif kondisi2:
    # Kode jika kondisi2 TRUE
elif kondisi3:
    # Kode jika kondisi3 TRUE
else:
    # Kode jika semua kondisi FALSE

Contoh: Nama Hari

Kode Program


hari = 3

if hari == 1:
    print("Senin")
elif hari == 2:
    print("Selasa")
elif hari == 3:
    print("Rabu")
elif hari == 4:
    print("Kamis")
else:
    print("Hari tidak valid")

Output Terminal

Rabu

Operator Logika

Operator Deskripsi Cara Kerja
and DAN TRUE jika keduanya TRUE
or ATAU TRUE jika salah satu TRUE
not TIDAK Membalik nilai (True jadi False)
Truth Table (Tabel Kebenaran):
AND
T and T = T
T and F = F
F and T = F
F and F = F
OR
T or T = T
T or F = T
F or T = T
F or F = F
NOT
not T = F
not F = T

Contoh Operator Logika


x = 10

# AND
print(x > 5 and x < 15)   # True (keduanya True)
print(x > 15 and x < 20)  # False (salah satu False)

# OR
print(x > 5 or x < 5)     # True (satu True)
print(x > 15 or x < 5)    # False (keduanya False)

# NOT
print(not(x > 5))         # False
print(not(x < 5))         # True

Nested IF (IF Bersarang)

Penjelasan

Kita bisa memasukkan blok IF di dalam blok IF lainnya.

Digunakan untuk mengecek kondisi bertingkat atau spesifik.

Format Kode


if kondisi1:
    if kondisi2:
        # Kode jika kondisi1
        # DAN kondisi2 TRUE
    else:
        # Kode jika kondisi1 TRUE
        # tapi kondisi2 FALSE
else:
    # Kode jika kondisi1 FALSE

Contoh Nested IF

Kode Program


nilai = 85
absensi = 90

if nilai >= 70:
    # Lulus secara nilai, cek absensi
    if absensi >= 80:
        print("Grade: A")
    elif absensi >= 60:
        print("Grade: B")
    else:
        print("Grade: C")
else:
    print("Tidak Lulus")

Output Terminal

Grade: A

Kombinasi Logika

Validasi Kelayakan Pemilih Pemilu: Usia >=17 atau Sudah Menikah

Kode Program


usia = 16
menikah = True

if usia >= 17 or menikah:
    print("Bisa MEMILIH")
else:
    print("Belum memenuhi syarat")

Output Terminal

Bisa MEMILIH

Tips Menulis Kondisi

  1. Mulai dari kondisi yang paling umum/besar.
  2. Gunakan elif untuk kondisi yang saling lepas (mutually exclusive).
  3. Gunakan Nested If untuk kondisi yang bergantung (dependent).
  4. Tanda Kurung: Gunakan () untuk memperjelas prioritas logika.
if (a > 5 and b < 10) or (a < 3 and b > 7):

Panduan Pengerjaan Tugas

Untuk memastikan pemahaman kondisi majemuk, ikuti langkah-langkah berikut:

  • Analisis Kondisi: Identifikasi kondisi yang menggunakan and/or, dan urutan if-elif-else.
  • Desain Alur: Buat pseudocode dengan nested if jika diperlukan.
  • Implementasi: Tulis kode dengan komentar jelas, gunakan operator logika dengan benar.
  • Testing: Jalankan dengan berbagai input untuk memastikan semua kondisi bekerja.
  • Refleksi: Tambahkan cell teks untuk refleksi penggunaan kondisi majemuk.
  • Dokumentasi: Jelaskan logika kode dan hasil di komentar.
