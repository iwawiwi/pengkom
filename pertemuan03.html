Apa itu Percabangan?

Percabangan adalah struktur kontrol yang memungkinkan program membuat keputusan berdasarkan kondisi tertentu.


Program melakukan hal berbeda berdasarkan kondisi yang berbeda!

üö¶ Lampu Lalu Lintas

üî¥ MERAH → BERHENTI

üü° KUNING → PERSIAPAN

üü¢ HIJAU → JALAN

Kapan Membutuhkan Percabangan?

  • ‚úÖ Memvalidasi input user
  • ‚úÖ Mengambil keputusan berdasarkan nilai
  • ‚úÖ Menangani berbagai kasus
  • ‚úÖ Mengontrol alur program

Contoh Real:

  • Jika nilai >= 60, Lulus
  • Jika usia < 17, No KTP
  • Jika saldo cukup, Transaksi OK
  • Jika hujan, Bawa Payung

Operator Perbandingan

Operator Arti Contoh Hasil
== Sama dengan 5 == 5 True
!= Tidak sama 5 != 3 True
> Lebih besar 5 > 3 True
< Lebih kecil 5 < 3 False
>= Lebih besar sama dengan 5 >= 5 True
<= Lebih kecil sama dengan 5 <= 3 False

Contoh Perbandingan


a = 10
b = 5

print("a == b:", a == b)    # False
print("a != b:", a != b)    # True
print("a > b:", a > b)      # True
print("a < b:", a < b)      # False
print("a >= b:", a >= b)    # True
print("a <= b:", a <= b)    # False

Struktur IF

Blok kode hanya dijalankan jika kondisi bernilai TRUE.


if kondisi:
    # Kode dijalankan
                            

nilai = 75

if nilai >= 60:
    print("Lulus!")

# Output: Lulus!

Struktur IF-ELSE

Jika kondisi TRUE, jalankan blok IF.
Jika FALSE, jalankan blok ELSE.


angka = 7

if angka % 2 == 0:
    print("Genap")
else:
    print("Ganjil")
Output: Ganjil

Contoh: Grade Nilai (elif)


nilai = 85

if nilai >= 90:
    grade = "A"
elif nilai >= 80:
    grade = "B"
elif nilai >= 70:
    grade = "C"
elif nilai >= 60:
    grade = "D"
else:
    grade = "E"

print(f"Grade: {grade}")
Output: Grade: B

Contoh: Kalkulator Sederhana


operasi = "+"
a, b = 10, 5

if operasi == "+":
    hasil = a + b
elif operasi == "-":
    hasil = a - b
elif operasi == "*":
    hasil = a * b
elif operasi == "/":
    if b != 0:
        hasil = a / b
    else:
        hasil = "Error"
else:
    hasil = "Tidak valid"

print(f"Hasil: {hasil}")
