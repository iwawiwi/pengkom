Apa itu Percabangan?

Percabangan adalah struktur kontrol yang memungkinkan program membuat keputusan berdasarkan kondisi tertentu.


Program melakukan hal berbeda berdasarkan kondisi yang berbeda!

Kapan Membutuhkan Percabangan?

  • Memvalidasi input user
  • Mengambil keputusan berdasarkan nilai
  • Menangani berbagai kasus
  • Mengontrol alur program

Contoh Real:

  • Jika nilai >= 60, Lulus
  • Jika usia < 17, No KTP
  • Jika saldo cukup, Transaksi OK
  • Jika hujan, Bawa Payung

Operator Perbandingan

Operator Arti Contoh Hasil
== Sama dengan 5 == 5 True
!= Tidak sama 5 != 3 True
> Lebih besar 5 > 3 True
< Lebih kecil 5 < 3 False
>= Lebih besar sama dengan 5 >= 5 True
<= Lebih kecil sama dengan 5 <= 3 False

Contoh Perbandingan

Kode Program


a = 10
b = 5

print("a == b:", a == b)    # False
print("a != b:", a != b)    # True
print("a > b:", a > b)      # True
print("a < b:", a < b)      # False
print("a >= b:", a >= b)    # True
print("a <= b:", a <= b)    # False

Output Terminal

a == b: False
a != b: True
a > b: True
a < b: False
a >= b: True
a <= b: False

Struktur IF

Penjelasan & Format

Blok kode hanya dijalankan jika kondisi bernilai TRUE.

Format IF:

if kondisi:
    # Blok kode yang dijalankan
    # jika kondisi TRUE

Contoh Implementasi


nilai = 75

if nilai >= 60:
    print("Lulus!")
                        
Output: Lulus!

Penting: Indentasi di Python

Python menggunakan indentasi (spasi atau tab) untuk menentukan blok kode, bukan kurung kurawal { } seperti bahasa lain.

BENAR


if kondisi:
    print("Ini blok IF")
    print("Baris kedua")
else:
    print("Ini blok ELSE")

SALAH


if kondisi:
print("Indentasi salah!")  # Error!
else:
    print("OK")

Catatan: Gunakan 4 spasi atau tab untuk indentasi standar.

Struktur IF-ELSE

Penjelasan & Format

Jika kondisi TRUE, jalankan blok IF.
Jika FALSE, jalankan blok ELSE.

Format IF-ELSE:

if kondisi:
    # blok kode jika kondisi TRUE
else:
    # blok kode jika kondisi FALSE

Contoh Implementasi


angka = 7

if angka % 2 == 0:
    print("Genap")
else:
    print("Ganjil")
                        
Output: Ganjil

Struktur IF-ELIF-ELSE

Penjelasan & Format

Digunakan ketika ada lebih dari dua kondisi yang perlu diperiksa secara berurutan.

Format IF-ELIF-ELSE:

if kondisi1:
    # blok kode jika kondisi1 TRUE
elif kondisi2:
    # blok kode jika kondisi2 TRUE tapi kondisi1 FALSE
elif kondisi3:
    # blok kode jika kondisi3 TRUE tapi kondisi1 & kondisi2 FALSE
else:
    # blok kode jika semua kondisi FALSE

Contoh: Grade Nilai

Kode Program


nilai = 85

if nilai >= 90:
    grade = "A"
elif nilai >= 80:
    grade = "B"
elif nilai >= 70:
    grade = "C"
elif nilai >= 60:
    grade = "D"
else:
    grade = "E"

print(f"Grade: {grade}")

Output Terminal

Grade: B

Pikirkan: Jika kondisi dibalik (mulai dari if nilai >= 60 untuk D, lalu elif >=70 untuk C, dst.), apakah hasil sama untuk nilai 85?

operasi = "+"
