Dasar Matplotlib


import matplotlib.pyplot as plt

# Data
x = [1, 2, 3, 4, 5]
y = [10, 20, 25, 30, 50]

# Plotting
plt.plot(x, y)

# Aksesoris
plt.title("Contoh Grafik Kenaikan")
plt.xlabel("Bulan ke-")
plt.ylabel("Penjualan (Juta)")

# Tampilkan
plt.show()

Bar Chart (Diagram Batang)


kategori = ['A', 'B', 'C', 'D']
nilai = [85, 90, 75, 60]

plt.bar(kategori, nilai, color='teal')

plt.title("Nilai Rata-rata Kelas")
plt.show()

Gunakan plt.barh() untuk horizontal bar.

Scatter & Pie Chart

Scatter Plot


plt.scatter(tinggi, berat)
plt.show()

Pie Chart


size = [30, 20, 50]
label = ['X', 'Y', 'Z']
plt.pie(size, labels=label)
plt.show()

Intro to Seaborn üåä

Library di atas Matplotlib yang lebih cantik & mudah untuk statistik.


import seaborn as sns
import matplotlib.pyplot as plt

# Contoh Histogram dengan garis distribusi
data = [1, 2, 2, 3, 3, 3, 4, 4, 5]
sns.histplot(data, kde=True)

plt.show()
