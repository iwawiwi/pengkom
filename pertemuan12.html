Dasar Matplotlib

Komponen utama untuk membuat visualisasi data.

Struktur Dasar

  • import matplotlib.pyplot as plt
  • Siapkan data (list/array)
  • Pilih jenis plot (plt.plot(), plt.bar(), dll)
  • Tambahkan label dan judul
  • plt.show() untuk menampilkan

Fungsi Aksesoris

  • plt.title() - Judul grafik
  • plt.xlabel() - Label sumbu X
  • plt.ylabel() - Label sumbu Y
  • plt.legend() - Legenda
  • plt.grid() - Garis grid

Contoh: Line Plot

Grafik garis untuk menampilkan tren data seiring waktu.


import matplotlib.pyplot as plt

# Data penjualan bulanan
bulan = [1, 2, 3, 4, 5, 6]
penjualan = [10, 20, 25, 30, 50, 45]

# Line plot dengan marker
plt.plot(bulan, penjualan, marker='o')
plt.title("Grafik Penjualan Bulanan")
plt.xlabel("Bulan")
plt.ylabel("Penjualan (Juta)")
plt.show()

Membuat grafik garis dengan titik marker untuk setiap data point.

Bar Chart (Diagram Batang)

Membandingkan nilai antar kategori dengan batang vertikal.


import matplotlib.pyplot as plt

# Data kategori dan nilai
grade = ['A', 'B', 'C', 'D']
jumlah = [8, 25, 10, 2]

# Buat bar chart dengan warna
plt.bar(grade, jumlah, color='skyblue')
# Tambahkan judul
plt.title("Distribusi Nilai Mahasiswa")
plt.show()

Parameter color bisa menggunakan nama warna atau kode hex. Gunakan plt.barh() untuk horizontal.

Scatter Plot

Melihat hubungan/korelasi antara dua variabel numerik.


import matplotlib.pyplot as plt

# Data tinggi dan berat badan
tinggi = [150, 160, 165, 170, 175, 180]
berat = [45, 55, 60, 65, 70, 75]

# Scatter plot
plt.scatter(tinggi, berat, color='coral', 
            alpha=0.7)
# Label dan judul
plt.xlabel("Tinggi Badan (cm)")
plt.ylabel("Berat Badan (kg)")
plt.title("Hubungan Tinggi vs Berat Badan")
plt.show()

Parameter alpha mengatur transparansi titik (0-1). Berguna untuk data yang overlap.

Pie Chart

Menampilkan proporsi atau persentase dari keseluruhan data.


import matplotlib.pyplot as plt

# Data distribusi nilai
grade = ['A', 'B', 'C', 'D']
jumlah = [8, 25, 10, 2]

# Pie chart dengan persentase
plt.pie(jumlah, labels=grade, 
        autopct='%1.1f%%', 
        colors=['gold', 'lightblue', 
                'lightgreen', 'coral'])
# Judul grafik
plt.title("Distribusi Nilai Mahasiswa")
plt.show()

Parameter autopct menampilkan persentase otomatis. Gunakan explode untuk memisahkan slice.

Intro to Seaborn

Library statistik di atas Matplotlib dengan style yang lebih menarik.


import seaborn as sns
import matplotlib.pyplot as plt

# Data nilai mahasiswa
nilai = [75, 80, 85, 78, 92, 88, 90, 85, 82, 87]

# Histogram + KDE (Kernel Density Estimation)
sns.histplot(nilai, kde=True, color='skyblue', 
             alpha=0.7)

plt.title("Distribusi Nilai Mahasiswa")
plt.xlabel("Nilai")
plt.ylabel("Frekuensi")
plt.show()

Parameter kde=True menambahkan garis kurva distribusi dengan style yang otomatis.

Panduan Pengerjaan Tugas

Untuk memastikan pemahaman visualisasi data:

  • Subplot Layout: Gunakan plt.subplot() atau plt.subplots() untuk multiple charts.
  • Data Preparation: Siapkan data dalam format yang sesuai untuk setiap jenis chart.
  • Chart Customization: Tambahkan title, label, dan legend untuk setiap subplot.
  • Figure Management: Atur ukuran figure dan spacing antar subplot.
  • Testing: Jalankan kode dan pastikan semua chart terlihat dengan baik.
  • Dokumentasi: Jelaskan setiap langkah visualisasi di komentar kode.
