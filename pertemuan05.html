Apa itu Perulangan?

Perulangan (Loop) adalah struktur kontrol yang memungkinkan blok kode dieksekusi berulang kali.

TANPA Loop:
print("Hello")
print("Hello")
print("Hello")
print("Hello")
print("Hello")
DENGAN Loop: 

for i in range(5):
    print("Hello")

Lebih singkat, efisien, dan mudah diubah!

for Loop

Digunakan untuk mengulang kode sejumlah kali yang sudah diketahui atau mengiterasi item dalam sequence.

Syntax & Contoh


# Syntax
for item in iterable:
    # Kode yang diulang

# Contoh
for i in range(5):
    print(i)

Output Terminal

0
1
2
3
4

Fungsi range()

Fungsi range() menghasilkan urutan angka yang bisa digunakan dalam loop. Urutan ini tidak menyimpan semua angka di memori (efisien untuk angka besar).

Syntax Penjelasan Contoh Hasil
range(n) Dari 0 sampai n-1 range(5) 0, 1, 2, 3, 4
range(start, end) Dari start sampai end-1 range(2, 6) 2, 3, 4, 5
range(start, end, step) Dengan loncatan step range(0, 8, 2) 0, 2, 4, 6
range(start, end, -step) Hitung mundur range(10, 0, -2) 10, 8, 6, 4, 2

Variasi for Loop

Hitung Mundur


for i in range(5, 0, -1):
    print(i)
Output Terminal
5
4
3
2
1

Total Penjumlahan


total = 0
for i in range(1, 101):
    total += i
print(total)
Output Terminal
5050

while Loop

Digunakan untuk mengulang kode selama kondisi bernilai TRUE. Jumlah perulangan mungkin tidak diketahui di awal.


i = 0
while i < 5:
    print(i)
    i += 1  # UPDATE KONDISI!

Cocok untuk:

  • Menunggu input user yang valid
  • Game loop
  • Membaca file sampai habis

Peringatan: Infinite Loop

BENAR


i = 0
while i < 5:
    print(i)
    i += 1  # Ada update

SALAH (BAHAYA)


i = 0
while i < 5:
    print(i)
    # i tidak berubah
    # Loop selamanya!

break dan continue

break

Menghentikan loop secara paksa.


for i in range(10):
    if i == 5:
        break
    print(i)
# Output: 0, 1, 2, 3, 4

continue

Skip iterasi saat ini, lanjut ke berikutnya.


for i in range(5):
    if i == 2:
        continue
    print(i)
# Output: 0, 1, 3, 4

Contoh: Kalkulator Interaktif


while True:
    operasi = input("Operasi (+, -, q=quit): ")
    
    if operasi == 'q':
        print("Bye!")
        break  # Keluar dari while
    
    a = float(input("Angka 1: "))
    b = float(input("Angka 2: "))
    
    if operasi == '+':
        print(f"Hasil: {a + b}")
    elif operasi == '-':
        print(f"Hasil: {a - b}")

Ringkasan Hari Ini

  • for loop: Untuk iterasi yang jumlahnya jelas.
  • while loop: Untuk iterasi berbasis kondisi.
  • range(): Membuat urutan angka.
  • Flow Control: break (stop) dan continue (skip).
