Apa itu Perulangan?

Perulangan (Loop) adalah struktur kontrol yang memungkinkan blok kode dieksekusi berulang kali.

TANPA Loop:
print("Hello")
print("Hello")
print("Hello")
print("Hello")
print("Hello")
DENGAN Loop: 

for i in range(5):
    print("Hello")

Lebih singkat, efisien, dan mudah diubah!

for Loop

Digunakan untuk mengulang kode sejumlah kali yang sudah diketahui atau mengiterasi item dalam sequence.


# Syntax
for item in iterable:
    # Kode yang diulang

# Contoh
for i in range(5):
    print(i)
Output: 0 1 2 3 4

Fungsi range()

Syntax Deskripsi Contoh Hasil
range(n) 0 sampai n-1 [0, 1, 2, 3, 4] (n=5)
range(start, end) start sampai end-1 [2, 3, 4, 5] (2, 6)
range(s, e, step) Dengan loncatan step [0, 2, 4, 6] (0, 8, 2)

Variasi for Loop


# Hitung Mundur
for i in range(5, 0, -1):
    print(i)
5, 4, 3, 2, 1

# Total Penjumlahan
total = 0
for i in range(1, 101):
    total += i
print(total)
5050

while Loop

Digunakan untuk mengulang kode selama kondisi bernilai TRUE. Jumlah perulangan mungkin tidak diketahui di awal.


i = 0
while i < 5:
    print(i)
    i += 1  # UPDATE KONDISI!

Cocok untuk:

  • Menunggu input user yang valid
  • Game loop
  • Membaca file sampai habis

break dan continue

break

Menghentikan loop secara paksa.


for i in range(10):
    if i == 5:
        break
    print(i)
# Output: 0, 1, 2, 3, 4

continue

Skip iterasi saat ini, lanjut ke berikutnya.


for i in range(5):
    if i == 2:
        continue
    print(i)
# Output: 0, 1, 3, 4

Contoh: Kalkulator Interaktif


while True:
    operasi = input("Operasi (+, -, q=quit): ")
    
    if operasi == 'q':
        print("Bye!")
        break  # Keluar dari while
    
    a = float(input("Angka 1: "))
    b = float(input("Angka 2: "))
    
    if operasi == '+':
        print(f"Hasil: {a + b}")
    elif operasi == '-':
        print(f"Hasil: {a - b}")
