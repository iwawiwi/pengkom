Apa itu CSV?

CSV (Comma Separated Values) adalah format teks sederhana untuk menyimpan data tabular.

Banyak digunakan untuk pertukaran data antara aplikasi (Excel ↔ Python ↔ Database).

Contoh isi file data.csv:


Nama,Umur,Kota
Budi,20,Jakarta
Siti,21,Bandung
Andi,19,Surabaya

Membaca CSV dengan Pandas


import pandas as pd

# Load data
df = pd.read_csv('mahasiswa.csv')

# Inspeksi Data
print(df.head())      # 5 baris pertama
print(df.info())      # Tipe data & memory
print(df.describe())  # Statistik dasar

Data Cleaning: Missing Values

Data di dunia nyata seringkali kotor/kosong.


# Cek data kosong (True/False)
print(df.isnull().sum())

# Solusi 1: Hapus baris yang kosong
df_bersih = df.dropna()

# Solusi 2: Isi dengan nilai default (ex: 0)
df_isi = df.fillna(0)

# Solusi 3: Isi dengan Rata-rata
mean_umur = df['Umur'].mean()
df['Umur'] = df['Umur'].fillna(mean_umur)

Data Manipulation

Filtering


# Filter Umur > 20
senior = df[ df['Umur'] > 20 ]

# Filter Kota Jakarta
jkt = df[ df['Kota'] == 'Jakarta' ]

Sorting


# Urutkan berdasarkan Umur
df_urut = df.sort_values('Umur')

# Descending (Besar ke Kecil)
df_desc = df.sort_values('Umur', ascending=False)

Hands-On: Analisis Penjualan

Diberikan file transaksi.csv (Produk, Jumlah, Harga).

  1. Load file menggunakan Pandas.
  2. Buat kolom baru Total = Limit * Harga.
  3. Hitung Total Omset (Sum dari kolom Total).
  4. Tampilkan transaksi dengan Jumlah > 10.

Membersihkan Duplikat


# Cek jumlah duplikat
print(df.duplicated().sum())

# Hapus duplikat
df = df.drop_duplicates()

Penting agar analisis tidak bias karena data ganda.
