Apa itu Variabel?

Variabel adalah "wadah" atau "tempat" di memori komputer untuk menyimpan nilai/data.


Bayangkan sebuah kotak yang diberi label nama, dan kita bisa memasukkan isi apa saja ke dalamnya.

üì¶
"Budi"
nama (Label)

Membuat Variabel

Di Python, variabel dibuat secara otomatis saat kita memberikan nilai padanya (Assignment).

Tidak perlu deklarasi tipe data secara eksplisit!


nama = "Budi"
umur = 20
tinggi = 165.5
mahasiswa = True

print(nama)      # Budi
print(umur)      # 20

Aturan Penamaan Variabel

‚úÖ Boleh (Benar)

  • nama, nama_lengkap
  • nilai1, _kuota
  • Huruf, Angka, Underscore
  • Case Sensitive (nama ‚â† Nama)

‚ùå Tidak Boleh (Salah)

  • 1nama (Awal angka)
  • nama-lengkap (Pakai strip)
  • class, for, if (Keyword)
  • Pakai spasi

Operator Assignment

Digunakan untuk mempersingkat penulisan operasi aritmatika dan penyimpanan nilai kembali ke variabel.

Operator Contoh Sama Dengan
+= x += 3 x = x + 3
-= x -= 2 x = x - 2
*= x *= 4 x = x * 4
/= x /= 2 x = x / 2

Konversi Tipe Data (Casting)

Fungsi Built-in

  • int(): ubah ke integer
  • float(): ubah ke desimal
  • str(): ubah ke string

angka_str = "100"
# int(angka_str) + 5
print(int(angka_str) + 5)  # 105

nilai = 99.9
print(int(nilai))          # 99 (Pembulatan ke bawah)

harga = 25000
print("Rp " + str(harga))  # "Rp 25000"
