Pertemuan 14: Algoritma & Mini Project

Agenda Hari Ini

1. Briefing Project

Penjelasan tugas besar, aturan main, dan pembentukan kelompok.

2. Workshop Algoritma

Merancang alur program (Flowchart/Pseudocode) sebelum coding.

Pengumuman Final Mini Project

  • Tujuan: Membuat aplikasi analisis data sederhana menggunakan Python.
  • Tim: 4-5 Orang per kelompok.
  • Deadline: Pertemuan 16 (Presentasi).
  • Output: Kode Program, Slide Presentasi, Laporan Singkat.

Pilihan Topik Project

1. Data Akademik

Analisis nilai mahasiswa, kelulusan, statistik IPK.

2. Data Penjualan

Analisis transaksi toko, produk terlaris, omset bulanan.

3. Data Kesehatan

Analisis BMI, detak jantung, riwayat pasien (Dummy Data).

Wajib Ada (Requirements)

Input Process
  • Membaca File CSV
  • Validasi Input / Data Cleaning
Analitis
  • Minimal 3 Fungsi Analisis (Statistik, Filtering, dll)
  • Penggunaan Library (Pandas/Math)
Output
  • Minimal 1 Grafik (Matplotlib/Seaborn)
  • Export hasil ke CSV baru
Code Quality
  • Variabel deskriptif
  • Ada komentar penjelas
  • Modular (menggunakan Fungsi)

Rubrik Penilaian

Komponen Bobot
Kode Program (Fungsionalitas) 30%
Visualisasi Data & Kreativitas 20%
Penerapan Materi (Loop, If, List, dll) 15%
Kerjasama Tim & Dokumentasi 15%
Presentasi Akhir 20%
TOTAL 100%

Workshop: Merancang Algoritma

Sebelum coding, WAJIB membuat rancangan dulu!

MULAI
  Baca file "data.csv"
  JIKA file tidak ada -> Tampilkan Error
  LAIN ->

     Buat Grafik Bar
     Simpan "hasil.csv"
SELESAI

Pembentukan Kelompok

Sesi Diskusi (15 Menit)

  1. Bentuk kelompok 4-5 orang.
  2. Tentukan Ketua Kelompok.
  3. Pilih Topik Project.
  4. Submit Nama Tim & Topik ke Dosen.

Timeline Project

  • Pertemuan 14 (Hari Ini): Briefing & Design.
  • Pertemuan 15 (Minggu Depan): Quiz 2 + Progress Check (Asistensi).
  • Pertemuan 16 (Final): Presentasi & Demo.
